# VSSUT CSE APP

An cross plateform app which helps you with provodes the information regarding eduacational courses, daily life schedule, assignments, rest day , etc

# Features

Login/ Signup Page
Profile page
Timetable
Holiday List
Syllabus
Lecture Notes and Class Notes(with weekly update)
Contact Page
Feedback page
